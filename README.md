```
        [ ИНТЕРНЕТ ]

        Virtual_IP
    blnc-1      blnc-2
         (192.168.1.x)
     
            |
      ( eth0: NAT )
      [  ROUTER-1
         ROUTER-2  ]
      /     |     \
     /      |      \
    /       |       \
= DMZ =  = APP =   = DB =
(10.0.10.x) (10.0.20.x)  (10.0.30.x)
   |        |         |
 front-1    back-1     db-1
```
