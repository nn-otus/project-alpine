## Тестовый вариант - Развертывание высокодоступного отказоустойчивого веб-приложения

+ Схема начального стенда
```
[ ИНТЕРНЕТ ]
         blnc-1
         blnc-2
            |
      ( eth0: NAT )
      [  ROUTER-1
         ROUTER-2  ]
      /     |     \
     /      |      \
    /       |       \
= DMZ =  = APP =   = DB =
(10.0.1) (10.0.2)  (10.0.3)
   |        |         |
 front-1    back-1     db-1
 front-2    back-2     db-2
```
+ На этом этапе мониторинг не настраиваем, не хватает ресурсов
